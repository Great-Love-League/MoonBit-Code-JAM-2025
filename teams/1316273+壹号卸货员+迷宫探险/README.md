# 迷宫探险 v1.0.0 版本

## 游戏简介

**迷宫探险** 是一款由 **壹号卸货员** 开发的迷宫解谜游戏。游戏旨在通过有趣的迷宫挑战，提升玩家的逻辑思维能力和空间感知能力，同时提供一个充满挑战和乐趣的探险体验。

## 操作说明

1. **开始游戏**：点击“开始游戏”按钮启动游戏。
2. **游戏过程**：
   - 游戏开始后，进入倒计时，玩家开始移动。
   - 使用上下左右控制选项选择，目前只开发了下移和右移。

## 技术特色

- **探险模式**：由点及面的散发玩家思维快速进行思考，得到最优线路后进行移动。
- **复杂路线**：后期加入更复杂的地图考验玩家的反应能力。
