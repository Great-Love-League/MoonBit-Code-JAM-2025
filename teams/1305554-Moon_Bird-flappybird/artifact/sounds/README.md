# 音频文件说明

请将以下音频文件放入此目录：

## 必需的音频文件：

1. **jump.mp3** - 跳跃音效
2. **item.mp3** - 道具收集音效  
3. **score.mp3** - 得分音效
4. **gameover.mp3** - 游戏结束音效
5. **background.mp3** - 背景音乐（可选）

## 音频文件要求：

- 格式：MP3
- 采样率：44.1kHz 或 48kHz
- 比特率：128kbps 或更高
- 文件大小：每个音效文件建议小于100KB

## 使用方法：

将音频文件放入此目录后，刷新游戏页面即可听到音效。

如果没有音频文件，游戏会自动使用程序生成的音效。
