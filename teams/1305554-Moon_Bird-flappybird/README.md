# Flappy Bird 增强版 - WebAssembly版本

一个基于MoonBit语言开发的Flappy Bird增强版游戏，使用WebAssembly技术实现高性能的游戏体验。

## 🎮 游戏特色

### 核心玩法
- **经典Flappy Bird玩法**：控制小鸟避开管道，获得高分
- **多种游戏模式**：经典、限时、生存、困难、挑战模式
- **形态进化系统**：随着分数增长，小鸟会进化成不同形态
- **丰富道具系统**：12种不同道具带来独特游戏体验，包含buff和debuff道具

### 形态进化系统
| 形态 | 解锁分数 | 特殊能力 | 属性加成 |
|------|----------|----------|----------|
| 🐣 雏鸟 | 0分 | 学习加速 | +20%经验获取 |
| 🐦 成鸟 | 5分 | 平衡发展 | 无特殊能力 |
| 🦅 雄鹰 | 10分 | 鹰眼预警 | 跳跃力+2，生命+1 |
| 🔥 凤凰 | 15分 | 涅槃重生 | 跳跃力+4，生命+2，60%概率涅槃重生 |
| 🐉 神龙 | 50分 | 重力控制 | 跳跃力+5，生命+4 |

## 🛠️ 技术实现

### 开发技术栈
- **MoonBit**：主要游戏逻辑开发语言
- **WebAssembly**：高性能游戏运行环境
- **JavaScript**：前端界面和渲染
- **HTML5 Canvas**：游戏画面渲染

### 核心架构

#### 游戏状态管理
```moonbit
pub struct State {
  mut count : Int
  mut gameover : Bool
  mut game_started : Bool
  mut score : Int
  mut pipes : Array<Pipe>
  mut pipe_speed : Int
  mut game_mode : GameMode
  mut evolution_stage : EvolutionStage
  mut player_stats : PlayerStats
  mut items : Array<Item>
  mut active_effects : Array<ItemEffect>
}
```

#### 道具系统
- **道具生成**：50%概率在管道后生成道具
- **碰撞检测**：精确的矩形碰撞检测
- **效果管理**：时间管理和效果叠加控制
- **视觉反馈**：闪烁效果和状态显示

#### 形态进化
- **分数触发**：基于分数自动进化
- **属性加成**：每阶段提供不同能力
- **视觉变化**：不同形态的独特外观
